## Что делаем

Подключаем карту «План города Тифлиса, 1904 г.» как новый слой исторической подложки. Тайлы XYZ уже привязаны (zoom 12–17), границы — bounds из metadata. Цель — слой появляется в админке (редактор координат) и доступен на странице `/tbilisi` рядом с картой 1898.

## Шаги

1. **Тайлы** — распаковать архив в `public/tiles/tbilisi-1904/{z}/{x}/{y}.png` (zoom 12–17, ~700 файлов из `Map 1904.zip`). Это статика, в git попадает как есть.

2. **Регистрация слоя** в `src/lib/tbilisi-historical.ts`:
   - Добавить экспорт `TBILISI_1904: HistoricalConfig` с `kind: "tiles"`, `tiles: "/tiles/tbilisi-1904/{z}/{x}/{y}.png"`, `minzoom: 12`, `maxzoom: 17`, attribution из metadata.json: *«План города Тифлиса, 1904 г., изд. Строительным отделением городской управы по указаниям инж. Т. Энфиаджианца · Национальный архив Грузии»*.
   - Добавить запись в массив `HISTORICAL_MAPS`: `{ id: "1904", title: "Тифлис, 1904 (Городская управа)", year: 1904, config: TBILISI_1904, notes: "..." }`. Этого достаточно, чтобы слой сразу появился в выпадающем списке админки (`TbilisiCoordEditorPanel`) с фильтрацией точек по году ≤1904.

3. **Страница `/tbilisi`** — сейчас жёстко использует `TBILISI_1898`. Чтобы 1904-й тоже был виден:
   - В `TbilisiMap.tsx` заменить прямые ссылки на `TBILISI_1898` на выбор из `HISTORICAL_MAPS` по новому пропу/состоянию `historicalId` (default `"1898"`).
   - В панели исторического слоя (где сейчас тумблер + слайдер прозрачности) добавить компактный `Select` со списком доступных карт (только те, у кого `config !== null`): «1898» и «1904». Год для лейбла маркера (`HIST_YEAR`) брать из выбранной записи.
   - Сохранить выбор в URL search-параметре `hm` (id карты) в `src/routes/tbilisi.tsx` через ту же zod-схему, что и `h`/`o`/`d`, чтобы ссылка/перезагрузка не теряли выбор.

4. **Границы полицейских участков 1904 г.** — пока пропускаем (на скане они есть, но GeoJSON для них не делаем в этой задаче; добавим отдельным шагом, когда оцифруете). Поле `districtsUrl` для записи 1904 оставляем пустым — слой границ просто не показывается.

## Проверка

- `/admin` → вкладка Тбилиси → выбрать «Тифлис, 1904» в выпадающем списке карт → тайлы грузятся без 404, по умолчанию видны точки с `startYear ≤ 1904`, кнопка «По году карты (≤1904)» работает.
- `/tbilisi?h=1&hm=1904` → отображается карта 1904 г., слайдер прозрачности и тумблер участков продолжают работать; переключение `hm=1898 ↔ 1904` мгновенное.
- Атрибуция MapLibre внизу карты содержит правильный текст 1904 г.

## Технические заметки

- Файлы тайлов не модифицируются — копируются как есть; формат `{z}/{x}/{y}.png` совпадает с тем, что ожидает MapLibre raster-source.
- Старая логика `HIST_YEAR_1898 = 1898` в `TbilisiMap.tsx` (строка 35) становится производной от выбранной карты — это влияет на текстовые лейблы «существовала к 1898», их обобщим до «к {year}».
- Никакие данные церквей не меняем; точки остаются те же, фильтр по году уже умеет работать с любым `histYear`.
